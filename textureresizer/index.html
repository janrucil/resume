Texture resizer

Texture Resizer is a custom tool that saves me tons of time when working with image files.


It´s great for texture optimalization in Unreal Engine, allowing me to drag-drop multiple files at once and process them simultaneously.


Let´s say i just imported a few materials from Megascans, containing 1024x1024 textures each. But want them in 512x512.


All I need to do is export the .uasset files as images, run them through the tool, and reimport the output images into UE again - instead of opening each file in photoshop!



Whole folders of textures can be optimized in a matter of seconds. Each output file keeps the name of its source. The results are high quality and are identical with doing the same procedure in Photoshop.


It can also be used for the creation of appicons or website thumbnails


Written in C# (WPF)


You can download Texture Resizer for free here (Google Drive) and try yourself.
